## 🕹️ What is CyberNet?

**CyberNet: Hacker Terminal** is a fast-paced cyberpunk hacking game set in the darknet. You play as **GHOST**, an elite hacker running ops from a glitchy CRT terminal. Complete missions, fend off incoming cyber-attacks, earn credits, and upgrade your rig before your system integrity hits zero.

---

## 🎮 How to Play

| Action | Description |
|---|---|
| **Active Missions** | Pick a mission from the left panel and type the command in the terminal |
| **Incoming Threats** | Respond to attacks in the right panel before they drain your integrity |
| **Earn Credits ₿** | Complete missions to rack up credits |
| **Buy Upgrades ↑** | Spend credits on powerful upgrades to stay ahead |
| **Survive** | Watch your INTEGRITY bar — if it hits 0, it's game over, ghost |

## ✨ Features

- 🖥️ **Retro CRT aesthetic** — scanlines, glitch effects, and neon colors
- ⚡ **Real-time threat system** — incoming attacks that demand immediate action
- 📈 **XP & leveling** — gain experience and level up as you hack
- 🛒 **Upgrades shop** — spend your earnings on terminal upgrades
